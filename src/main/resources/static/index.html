AppForge

Микросервис AppForge позволяет шаблонизировать процесс разработки веб-приложений.

Используя уже наработанные и протестированные модули, он "куёт" полноценные представления для клиентских микросервисов согласно указанной ими JSON-конфигурации.

Всё, что потребуется, чтобы начать пользоваться AppForge — умение создавать эндпоинты (контроллеры) на стороне сервера, с которым будет происходить обмен данными в реальном времени.
